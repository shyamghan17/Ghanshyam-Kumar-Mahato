FUEL LEVEL INDICATOR

4.1 OBJECTIVE: To understand the use of NOT gate in a fuel level indicator application.

4.2 INTRODUCTION

The NOT gate has only one input and one output. It provides an output that is not the same as the input i.e. it changes a 1 at its input to a 0 at the output and vice-versa. The Boolean expression for NOT is expressed as a bar or a single prime or apostrophe over the variable. If input A passes through two NOT gates, the result is 'double NOT A', which equals the input A. The NOT operation is called a unary operation since it operates upon only one variable. Conversely, the OR and AND operations are binary operations as two or more variables are involved.

4.2.2 APPLICATION: FUEL LEVEL INDICATOR

To indicate whether the fuel level in a tank has reached the minimum (reserve) level, a simple NOT gate along with a digital output level sensor can be used.

4.2.3 CONCEPT:

When the fuel level drops below the reserve/minimum level (1/4 th level of tank), the level- sensor gives output as logic HIGH.  The NOT gate produces a LOW output, and the RED LED glows to indicate this fuel shortage status. To simulate the variation in the fuel level in the tank, a slider is provided.

        

Figure 1 Concept of Fuel Level Indicator

4.3 PROCEDURE

1.     Vary the slider upwards. This condition simulates that the fuel level is sufficient.

2.     Observe the LED output.

3.     Vary the slider below the minimum/reserve marking.

4.     Observe whether the drop in fuel level is indicated by the glowing of the RED LED.
