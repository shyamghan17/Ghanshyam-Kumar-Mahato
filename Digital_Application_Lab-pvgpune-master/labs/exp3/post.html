Q1. When a Logic 1 is applied on the input of a NOT gate, the output is ______.
A  Logic 0
B  Logic 1
C  High Impedance (Z)
D  None of these

Ans is A


Q2. If the 8-bit input is set to 1111 0000, the output of the 1’s complement circuit is ___________.

B  0000 0000
C  0000 1111
D  1010 0101

Ans is C
