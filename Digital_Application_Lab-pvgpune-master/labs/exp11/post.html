Q1. On comparing the AOI and XOR gate logic implementation, XOR logic results in a lower gate count and a reliable system. State TRUE or FALSE.
A  True
B  False

Ans is A


Q2. A certain system contains two identical circuits operating in parallel. As long as both are operating properly, the outputs of both circuits are always the same. If one of the circuits fails, the outputs will be at opposite levels at the same time. Which gate would you choose to detect that a failure has occurred in one of the circuit?
A  AND
B  OR
C  XOR
D  NOT

Ans is C


Q3. If one of the inputs of a 2-input XOR gate is tied to Vcc (+5 V), the XOR behaves as:
A  AND
B  NOT
C  OR
D  None of these

Ans is B


Q4. If one of the inputs of a 2-input XOR gate is connected to ground (0 V), the XOR:
A  passes the other input to the output with some delay.
B  blocks the other input from reaching the output.
C  inverts the other input
D  None of these.

Ans is A


Q5. Can XOR gate be used to compare the parity (no. of 1’s) of the input applied?
A  Yes
B  NO

Ans is A
