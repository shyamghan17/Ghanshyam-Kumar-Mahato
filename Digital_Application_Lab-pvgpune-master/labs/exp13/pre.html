Q1. A half adder is a circuit that adds two binary numbers and produces a sum(S) and a carry(CY). Which of the following is the correct way of expressing these outputs?
A  S = A XNOR B, CY = AB
B  S = AB, CY = A+B
C  S = A XOR B, CY = A+B
D  S = A XOR B, CY = AB

Ans is D


Q.2 A one-bit multiplier circuit produces an output Y:
A  Y = A+ B
B  Y = A.B
C  Y = A XOR B
D  Y = (AB)’

Ans is B


Q3. The expression for sum (S) of a full adder circuit is given by:
A  Y = ABC
B  Y = A XOR B XOR C
C  Y = A+B+C
D  None of these

Ans is B


Q4. To design a one bit subtractor using AOI logic, the Difference output (Diff) and borrow output (Br) is given by expressions:
A  Diff = A XNOR B; Br = AB
B  Diff = AB; Br = A’B
C  Diff = A+ B; Br = AB
D  Diff = A XOR B; Br = A’B

Ans is D


Q 5. While designing a Full Adder, two implementation options are available: Method 1: AOI Logic; Method 2: NAND Logic. Which of these methods provides a simple, compact and reliable design?
A  AOI Logic
B  NAND Logic
C  Both A and B
D  None of these

Ans is B
